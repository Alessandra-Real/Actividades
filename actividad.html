Ejemplo de Applet con Java

Un applet conforme a lo que encontramos en las páginas de Oracle: "es una aplicación hecha en Java que se llama desde un documenton HTML y se ejecuta dentro de un navegador". Es necesario recordar que el navegador dene tener instalado el plugin de la Máquina Virtual de Java (JVM)


Máquina Virtual de Java: Es el programa que se encarga de ejecutar el bytecode generado por el compilador de Java


Para entender la forma een que se manejan los applets vamos a ver un ejemplo que presenta el clásico Hola mundo:


Como puede observarse la única variable que utiliza este applet es g que representa un objeto de la clase Graphics sobre el cual vamos a dibujar la cadena de texto que resulta de ejecutar el applet


Para ejecutarlo es necesario mandarlo a llamar desde una página HTML, utilizando la etiqueta "applet" o la etiqueta "object". En el primer caso el archivo .class se declara en el atributo code, en el segundo se declara un elemento param donde sus atributos name serán code y value será el nombre del applet. Cabe mencionar que en el segundo caso es necesario declarar otros elementos param adicionales y en elemento object declara algunos parámetros como classid, codebase y type. En ambos casos es necesario declarar el tamaño de la ventana del applet con los parámetros width y height


La salida del programa será: Hola mundo!!


Todos sabemos que este es un ejemplo muy sencillo y lo habremos visto ya bastantes veces, probablemente cuando Dennis N. Ritchie lo escribió en lenguaje C. jamás imaginó que sería utilizado como ejemplo inicial de tantos lenguajes de programación
