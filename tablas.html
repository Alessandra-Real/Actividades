La palabra inglesa "FRAMEWORK" (Marco de trabajo) define, en terminos generales, un conjunto estandarizado de conceptos, prácticas y criterios para enfocar un tipo de problemática particular que sirve como indole similar.


En el desarrollo de Softwawe, un framework o infraestructur, es una estructura conceptual tecnologica de soporte definido, normalmente con artefactos o modulos de software concretos, que puede servir de base para la organización y desarrollo de software, tipicamente hablando, puede incluir soporte de programas, bibliotecas y un lenguaje interpretado, entre otras herramientas, para así poder ayudar a desarrrollar y unir componentes de un proyecto.

Apache Cocoon Usualmente llamado simplemente cocon, es un framework para aplicación web contruido en torno a los conceptos pipoline, encapsulación y desarrollo de aplicaciones web basado en componentes. Permite la rápida publicación de contenidos o una variable de formatos, entre los que se incluyen HTML, PHP y WML.
Apache Struts Struts se basa en el patrón de arquitectura de sorftware Modelo.Vista-Controlador (MVC) el cual es utilizado ampliamente y es considerado de gran solidez, El procesamiento se separa en tres secciones diferenciadas llamadas el modelo, las vistas y el controlador.
Google Guice Es un framework de inyección de independencias que puede ser utiliadas en app hechas con java donde la relación o dependencia entre objetos de negocio necesita ser administrado o mantenida.
JADE Java Agent Development framework o JADE es una plataforma software de desarrollo de agentes, implementada en Java
Java Pesistence API(JPA) Es un framework del lenguaje de programación Java que maneja datos relacionales en aplicaciones usando la Plataforma Java en sus ediciones Standard (Java SE) y Enterprise (Java EE). El objetivo que persigue el diseño de esta API es no perder las ventajas de la orientación a objetos al interactuar con una base de datos
